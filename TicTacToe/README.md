# 🎮 Tic Tac Toe Game (C++)

A console-based **Tic Tac Toe game** developed using C++.  
This project demonstrates **function-based programming, input validation, game logic, and scoreboard management**.

## 🚀 Features

- ✅ Two Player Mode
- ✅ Player Name Input
- ✅ X / O Symbol Selection
- ✅ Random Toss System
- ✅ Interactive 3×3 Game Board
- ✅ Winner Detection
- ✅ Draw Detection
- ✅ Scoreboard System
- ✅ Reset Scoreboard Option
- ✅ Input Validation
- ✅ Play Again Option

## 🛠️ Technologies Used

- C++
- Functions
- Conditional Statements
- Loops
- Arrays
- Random Number Generation

## 🎯 Game Rules

1. Two players play against each other.
2. Player 1 chooses X or O.
3. Toss decides who starts first.
4. Players select positions from **1 to 9**.
5. The first player to complete a row, column, or diagonal wins.
6. If all positions are filled without a winner, the match is declared a draw.
